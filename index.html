Simulation Office

Ein von Grund auf in C geschriebene, grafische Simulationssoftware die den Arbeitsverlauf in verschiedenen Abteilungen und ihre Abhängigkeit voneinander simulieren kann. Mithilfe der Simulation die sich auf statistische Daten und Verteilungsmuster stützt werden Enngpässe erfasst und in einem Bericht mitsamt Optimierungsvorschlägen ausgegeben.
Leider kann ich den Code hierfür nicht Opensource machen, für mehr Infos an mich wenden.

AlteredCoal Minecraft Mod

Modifizierung des populären Spiels "Minecraft".
Beinhaltet simple Physiksimulationen bei Fahrzeugen, Online-Multiplayer Daten synchronisierung (Client-Server Packetmanagement) und OpenGL Renderung. Code ist opensource und erhält regelmäßig Updates auf GitHub.
